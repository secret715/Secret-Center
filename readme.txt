Secret Center

內建聊天室、論壇、通知以及後臺管理的會員系統

官方網站：https://center.gdsecret.net/

-----------------------------------------------

安裝

1. 上傳文件後，執行 install.php
2. 填入相關資料
3. 安裝成功！

-----------------------------------------------

使用

* 後台位置：你的網址/admin/index.php
* 網站設定：網站後台 或 config.php
