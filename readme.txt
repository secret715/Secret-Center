Secret Center

內建聊天室、論壇、通知以及後臺管理的會員系統

官方網站：http://center.gdsecret.net/

-----------------------------------------------

安裝

1. 上傳文件後，執行 install.php
2. 填入相關資料
3. 安裝成功！

-----------------------------------------------

使用

* 後台位置：http://你的網址/admin/index.php
* 詳細資料(如：網站名稱)：config.php

-----------------------------------------------

從Secret會員系統8.1升級

1. 備份 Connections/SQL.php 及資料庫
2. 上傳新版，不要覆蓋 Connections/SQL.php
3. 執行 upgrade.php
4. 升級成功！
